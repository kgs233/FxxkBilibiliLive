# FxxkBilibiliLive

这是一个输出设备的双声道声音合并到另一个输出设备的左右两声道的其中一个声道中，以解决Bilibili Live中没有白名单的ASMR主播被迫使用单声道进行直播的问题

## 如何使用

你首先需要安装虚拟声卡，推荐使用VB-Audio Virtual Cable

然后重启电脑

在此项目中运行dotnet run（或者你也可以编译一下然后运行）

只支持Windows
